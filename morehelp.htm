VIEWu Additional Features

Background Pictures

FB:color, picture

FB:color filepath/filename.ext

Add a background picture

  • color = Font color: css_color, #rrggbb, #rgb, .colorclass (see end)
  • use relativepath/filename
  • .ext = Picture files: .jpg, .jpeg, .gif, .png, .bmp

.ColorClasses

These are classes declared in VIEWu that can be used for 'FB:color,...'
They provide outlined text for viewing text over a picture.
